Kufar Price Helper

Легкое Chromium-расширение, которое добавляет примерный USD-эквивалент к ценам BYN на Kufar и помогает отслеживать выбранные объявления.

GitHub Установка

Что умеет

  • Конвертирует BYN в USD по официальному курсу НБРБ.
  • Кэширует курс на 6 часов.
  • Работает с динамически обновляемыми страницами Kufar.
  • Позволяет включать/выключать расширение из popup.
  • Хранит выбранные объявления и историю цен локально в браузере.

Установка из исходников

  1. Скачайте репозиторий или ZIP-архив релиза.
  2. Откройте chrome://extensions или edge://extensions.
  3. Включите Developer mode.
  4. Нажмите Load unpacked и выберите папку расширения.
