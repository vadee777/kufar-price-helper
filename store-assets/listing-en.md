# Chrome Web Store / Microsoft Edge Add-ons listing: EN

## Name

Kufar Price Helper

## Short description

Shows approximate USD prices next to BYN prices on Kufar.

## Full description

Kufar Price Helper helps compare Kufar listings faster by adding an approximate USD equivalent next to prices shown in BYN.

Features:

- automatic BYN to USD conversion using the official NBRB exchange rate;
- 6-hour exchange-rate cache;
- support for dynamically updated Kufar pages;
- global controls from the extension popup;
- small optional support links in the popup;
- tracking for selected Kufar listings;
- weekly price-history snapshots stored locally in the browser.

The extension does not replace bank exchange rates and displays approximate reference prices only.
