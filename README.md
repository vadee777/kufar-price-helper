# Kufar Price Helper

Kufar Price Helper is a lightweight browser extension that converts Kufar prices from BYN to USD using the official NBRB exchange rate.

## Features

- Automatic BYN to USD conversion
- Official NBRB exchange rate
- Price history tracking
- Weekly price dynamics
- Lightweight browser experience
- No registration required
- No personal data collection

## Supported websites

- kufar.by

## Privacy

The extension does not collect, transmit, or share personal user data. Price history and settings are stored locally in the user's browser.
