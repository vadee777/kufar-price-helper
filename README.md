# Kufar Price Helper

Расширение для Chrome, Microsoft Edge и других Chromium-браузеров, которое показывает примерную цену в USD рядом с ценами BYN на Kufar и помогает отслеживать выбранные объявления.

## Features

- Пересчитывает цены BYN в USD по официальному курсу НБРБ.
- Кэширует курс на 6 часов.
- Работает с динамически обновляемыми страницами Kufar.
- Позволяет включать и отключать расширение из popup.
- Добавляет кнопку отслеживания рядом с пересчитанной ценой.
- Хранит выбранные объявления и недельные снимки истории цен в `chrome.storage.local`.
- Показывает компактное окно с историей цены.
- Не дублирует USD-метки при повторной отрисовке страницы.
